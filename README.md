# Villa

This repository is a rewrite of the original Villa that I created for 26C3 or so.

For a description of the Villa, listen to https://freakshow.fm/fs256-physisch-virtuell

# Building

## MacOS

### re

- `cd third_party/re`
- `make`
- `make install`

### rem

- `cd third_party/rem`
- `make`
- `make install`

### re

- `cd third_party/baresip`
- `make`
- `make install`
